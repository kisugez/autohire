# Autohyr

AI-powered recruitment automation platform built with Next.js 14.

## Features

- Candidate Management
- Job Postings & Tracking
- Visual Pipeline Management
- Workflow Automations
- Email Outreach Campaigns
- Analytics & Insights
- Platform Integrations

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- TailwindCSS
- Radix UI
- React Query
- React Flow

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
